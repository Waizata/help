Télécharger l'application

Pour télécharger l'application Waizata il vous suffit de cliquer sur le bouton "Télécharger" ci-dessous.
Notez cependant que l'application Waizata ne fonctionne que sur les téléphones Android.

  • Télécharger

Installation

  • Autoriser l'installation d'apk

    Allez dans les paramètres de votre téléphone portable afin d'autoriser l'installation d'application à partir de sources inconnues. Pour cela Allez dans Paramètres, puis Sécurité et cochez Sources inconnues

  • Trouvez le fichier ".apk"

    Parcourez le stockage interne de votre téléphone afin de trouver le fichier waizata.apk. Cliquez dessus afin de lancer l'installation

  • Lancer l'application

    Une fois l'installation terminée avec succès, retournez sur votre page d'accueil et cliquez sur l'application Waizata

Aide supplémentaire

Nous espérons que l'installation s'est bien passée et que vous pouvez dès à présent profiter pleinement des fonctionnalités que vous offre l'application Waizata mobile !

Cependant si vous avez des difficultés à installer l'application ou durant son utilisation n'hésitez pas à contacter notre équipe. Dans le cas de l'installation de l'application nous pouvons vous proposer des tutoriels plus complets afin de résoudre les problèmes que vous pourriez rencontrer
